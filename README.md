### **日省吾身吧台**

#### **想法的起源**

  每到年底人们都会发微博记录这一年来干的事情，算是对前一年的总结和新一年的展望，有的一年瘦了10斤，有的一年成了羽毛球高手，有的一年职位title升了一级，当看到这些微博的时候，我就想起每年屈指可数的回家跟亲人相聚的次数。转念再想，既然已经是事实了，我得记录下我具体是拿亲人相聚的时间(空间)换取的什么东西？如是有了这个想法，每天记录几件大事，到年底自动统计就出来了，结果便一目了然。
